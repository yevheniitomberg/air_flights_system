## air_flights_system
This project is a site for booking flights
___
#### I'm using following technologies in this project:
- Spring Boot
- Spring MVC
- Spring Data JPA
- Spring Security
- Thymeleaf
- API
- BCrypt encoder
- Server: Tomcat
- Database: MariaDB
- HTML/CSS/JS
- amd.tools SMTP
- VPS (Linux Ubuntu 20.04)

___
#### External dependencies:
- Jackson Databind
- Google Zxing
